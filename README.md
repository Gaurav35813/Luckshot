# Luckshot
🎰 LuckShot is a modern lucky number guessing game built with Python and Streamlit.
Players purchase a virtual ticket, choose a number, and compete against randomly generated lucky numbers.
The game features a 10-second countdown timer, multiple prize tiers, and an interactive casino-style interface.
Custom CSS enhances the visual experience with animations and responsive design.
A fun project demonstrating Streamlit, session state management, and UI customization.
